O Javascript é atualmente a linguagem de programação mais utilizada no desenvolvimento frontend e surgiu como uma solução que possibilitasse a interatividade em páginas web. Na época em que foi inventado, não haviam linguagens de programação que fossem capazes de rodar diretamente nos navegadores dos usuários, fazendo com que fosse necessário enviar requisições aos servidores para qualquer manipulação de elementos de uma página web. Com seu advento, foi possível utilizar trechos de código orientados a eventos, que reagiam conforme o usuário interagia com a página, revolucionando a maneira que sites eram construídos, transformando páginas estáticas em dinâmicas e mudando completamente a relação usuário-internet.
